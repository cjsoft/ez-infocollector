# 关于部署
- 需要mysql，在coding.io演示环境下可以直接部署，但是需要绑定一个mysql服务
- 修改数据库链接信息，请在cjs.py的connect()中修改，或者直接添加环境变量VCAP_SERVICE，格式与coding.io演示平台的相同
- 启动命令行见Procfile
